# DruidMacros

## Cat

Cat Form 
```
#showtooltip Cat Form
/dismount
/stand
/cancelaura Predatory Strikes(Rank 3)
/cancelaura Leader of the Pack
/cancelform [noform:1/3]
/cast [form:1] Dire Bear Form
/cast [form:3] Cat Form
/cast [noform:3] Cat Form
```

Dash
```
#showtooltip Dash
/stand
/dismount
/cancelform [noform:1/3]
/cast [form:1] Dire Bear Form
/cast [noform:3] Cat Form
/cast Dash
```
